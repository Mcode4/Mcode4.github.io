About Spot Homes

Summary

Inspired by Airbnb, this site is a site where you can buy and sell spots. This was a earlier and basic site made to show how a site like this would function. Primarily how users could create, modify, and delete data they've created and not data they didn't.

This was a great start for me understanding the fundamentals and is definitely recommend if you're curious about how any site works. Much of code in this project is harder to ready or not as optimal as my current projects do to me getting the hang of the process but it still does what it's meant to.
