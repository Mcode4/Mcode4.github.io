About Me

I specialize in web development, crafting dynamic and user-friendly websites. With expertise across the full stack, I am passionate about coding and learning new things. Some of my key points listed below.

Stack

I primarily work with HTML, CSS, and JavaScript, which I consider the backbone of web development. These technologies enable me to create responsive, accessible, and visually appealing websites, and I value them as essential skills for building the web.

Api Control

I've experimented with and implemented restful api before. The one being a superhero api where I had fetched data, parsed it, and used it in an application. Similar to how the frontend gets data from the backend.

Code-to-Cloud

For my projects listed, which are sites, have been hosted to the internet with the help of Render. I've also had some experience with Docker, AWS, and Github Pages to add to that.

Systems Thinking

When I start a project, I like to get a feel of what the structure should be. Usually I keep frontend and backend on seperate folder and add in files that match the folder. Like HTML and pages in frontend and database in backend. Assets and documents has its own folders.
